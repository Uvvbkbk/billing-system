# 商用對賬單生成系統 - 本地使用指南

歡迎使用您的專屬商用對賬單生成系統！本指南將協助您在本地電腦上輕鬆安裝和運行此應用，無需複雜的代碼知識。

## 🚀 第一步：準備環境 (僅需一次)

本應用基於 Python 運行。如果您的電腦尚未安裝 Python，請按照以下步驟操作：

1.  **下載 Python**：
    *   訪問 Python 官方網站：[https://www.python.org/downloads/](https://www.python.org/downloads/)
    *   下載 **Python 3.8 或更高版本**的安裝包（建議選擇最新穩定版）。

2.  **安裝 Python**：
    *   運行下載的安裝包。
    *   **非常重要**：在安裝向導的第一個界面，請務必**勾選 "Add Python to PATH" (將 Python 添加到環境變量)** 選項。這將確保系統能正確找到 Python。
    *   然後點擊 "Install Now" 或 "Next" 完成安裝。

## 📦 第二步：獲取應用文件

我將會把所有應用文件打包給您。請將這些文件下載到您電腦上的一個**固定文件夾**中，例如 `D:\對賬單系統` 或 `~/Documents/對賬單系統`。請不要隨意移動這些文件，尤其是 `reconciliation.db`，因為它保存著您的所有數據。

您將會看到以下幾個文件：

*   `app.py`：應用程序的核心代碼。
*   `requirements.txt`：應用程序所需的依賴庫列表。
*   `templates/`：包含 `index.html` 網頁界面的文件夾。
*   `reconciliation.db`：**您的數據庫文件**。所有客戶信息和對賬單數據都保存在這裡。**請勿刪除或修改此文件！**
*   `start_windows.bat`：Windows 系統的一鍵啟動腳本。
*   `start_unix.sh`：macOS 或 Linux 系統的一鍵啟動腳本。

## ▶️ 第三步：啟動應用

### 對於 Windows 用戶：

1.  找到您保存應用文件的文件夾。
2.  **雙擊 `start_windows.bat` 文件**。
3.  一個命令行窗口會彈出，它會自動安裝必要的組件並啟動應用。
4.  您的默認瀏覽器會自動打開，並顯示對賬單系統的界面。
5.  **請勿關閉這個命令行窗口**，否則應用將停止運行。

### 對於 macOS 或 Linux 用戶：

1.  打開終端 (Terminal) 應用。
2.  使用 `cd` 命令進入您保存應用文件的文件夾，例如：
    ```bash
    cd ~/Documents/對賬單系統
    ```
3.  給予啟動腳本執行權限 (僅需一次)：
    ```bash
    chmod +x start_unix.sh
    ```
4.  運行啟動腳本：
    ```bash
    ./start_unix.sh
    ```
5.  您的默認瀏覽器會自動打開，並顯示對賬單系統的界面。
6.  **請勿關閉這個終端窗口**，否則應用將停止運行。

## 🛑 第四步：關閉應用

當您不再需要使用對賬單系統時，只需**關閉您在第三步中打開的命令行/終端窗口**即可。應用程序將會停止運行。

## 💾 數據持久化

您的所有數據（客戶信息和對賬項目）都安全地保存在 `reconciliation.db` 文件中。只要您不刪除這個文件，您的數據將會一直存在。

**建議您定期備份 `reconciliation.db` 文件，以防萬一。**

## ❓ 常見問題與故障排除

*   **瀏覽器沒有自動打開？**
    *   請手動在瀏覽器中輸入地址：`http://127.0.0.1:5000`。
*   **啟動腳本報錯？**
    *   請檢查 Python 是否正確安裝，並且在安裝時勾選了 "Add Python to PATH"。
    *   檢查您的網絡連接，確保可以下載依賴庫。
*   **數據丟失了？**
    *   請檢查 `reconciliation.db` 文件是否還在應用文件夾中。如果被刪除，數據將無法恢復，除非您有備份。
